# TeamWork-of-SoftWare——MiNote

#### 项目简介
本项目基于小米便签开源项目进行学习参考二次开发，无商用性质，主要目的如下：

1. 实现源码的分析构建，了解基础的架构设计；
2. 对部分初始功能实现完善；
3. TDOD
